What is box-modal?

In CSS, the box model refers to the layout structure of an element, which consists of four parts: content, padding, border, and margin. It defines how these parts interact to determine the total size and spacing of an element on the page.

What are the box-modal properties? (explain)

The CSS box model properties include: Content: The actual content of the box (e.g., text, images). Padding: Space between the content and the border, inside the element. Border: A line surrounding the padding (optional), separating the padding from the margin. Margin: Space outside the border, creating distance from other elements.

What is the d/f b/w padding and margin?

The difference between padding and margin is that padding is the space inside an element, between the content and its border, while margin is the space outside an element, between the border and surrounding elements.

What is d/f b/w outline and border?

The difference between outline and border is that border is part of the element's box model and affects layout, while outline is a line drawn outside the element's border that does not affect the layout or box size.
